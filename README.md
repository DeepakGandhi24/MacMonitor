\f0\b\fs36 \cf0 \strokec2 How the Agent Works\
\pard\pardeftab720\partightenfactor0

\f2\b0\fs26 \cf0 Mac boots\
  
\f7 \uc0\u8594 
\f2  launchd reads com.macmonitor.agent.plist\
  
\f7 \uc0\u8594 
\f2  starts agent.py automatically as a background daemon\
  
\f7 \uc0\u8594 
\f2  agent collects metrics every 5 minutes:\
      CPU usage \'b7 Memory \'b7 Disk \'b7 Battery \'b7 Network \'b7 Top processes\
  
\f7 \uc0\u8594 
\f2  POSTs JSON to dashboard Mac on port 9876\
  
\f7 \uc0\u8594 
\f2  if agent crashes 
\f7 \uc0\u8594 
\f2  launchd restarts it automatically\
\pard\pardeftab720\sa240\partightenfactor0
